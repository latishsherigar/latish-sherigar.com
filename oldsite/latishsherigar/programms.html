2. File Splitter 

 

About this software

As the name suggests, this is a file splitter application .A file splitter is an application which can cut or split any big file into small pieces .The program gives the user the choice to split the file in user defined size. Good software when you have to transfer large game files and you don't have a Internet connection for transferring them .Also you can attach this splitted files as an attachment and email it your friends with fast Upload times since many mail services restrict the size of the attachments to be uploaded to less than 1 MB. When you want to join the splitted files this program can join the splitted files and you have your original file in its full glory.

 The executables and source are available for this application. 

The source is avaliable at file-splitter.c and file-splitter_header.c
